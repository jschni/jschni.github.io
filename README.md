# Portfolio — GitHub Pages

This is a minimal static portfolio scaffold suitable for GitHub Pages.

Quick start

1. Initialize git and commit:

```bash
git init
git add .
git commit -m "Initial portfolio"
```

2. Create a GitHub repository. For a user/organization site name it `yourusername.github.io`.

3. Add remote and push:

```bash
git remote add origin https://github.com/yourusername/your-repo.git
git branch -M main
git push -u origin main
```

4. Enable GitHub Pages in the repository settings: choose `main` branch (root) or `gh-pages` if you prefer.

Notes

- Replace the placeholder content in `index.html` and `assets/css/styles.css` with your info.
- To use a custom domain, add a `CNAME` file with the domain name and configure DNS.
